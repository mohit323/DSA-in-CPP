# Linked List in C++

A complete implementation of **Linked List Data Structure** in **C++** from scratch. This repository focuses on understanding how linked lists work internally without relying on STL.

---

## 📌 About

This repository contains the implementation of various linked list operations written in C++. The goal is to build a strong understanding of pointers, dynamic memory allocation, and linked list manipulation through hands-on coding.

Everything is implemented manually to strengthen problem-solving skills and prepare for Data Structures & Algorithms interviews.

---

## ✨ Features

- Create a Linked List
- Display a Linked List
- Count Total Nodes
- Sum of All Elements
- Find Maximum Element
- Find Minimum Element
- Search an Element
  - Linear Search
  - Improved Linear Search (Move to Head)
- Insert Node
  - At Beginning
  - At Any Position
  - At End
- Delete Node
- Check if List is Sorted
- Insert in a Sorted List
- Remove Duplicate Nodes
- Reverse Linked List
  - Using Array
  - By Reversing Links (Iterative)
  - Recursive Reverse
- Concatenate Two Linked Lists
- Merge Two Sorted Linked Lists
- Detect Loop in Linked List
- Find Middle Node
- Circular Linked List
- Doubly Linked List
