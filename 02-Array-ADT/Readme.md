# Arrays

This folder contains my implementations and practice programs related to Arrays in C++. Arrays are one of the fundamental data structures and serve as the foundation for learning more advanced data structures and algorithms.

## Topics Covered

### Array ADT

* Creating an Array
* Displaying Elements
* Understanding Size and Length
* Dynamic Memory Allocation

### Insert

* Insert at the Beginning
* Insert at a Specific Index
* Insert at the End

### Delete

* Delete from a Specific Index
* Shifting Elements After Deletion
* Updating Array Length

### Search

* Linear Search
* Improved Linear Search
* Binary Search (for sorted arrays)

### Reverse

* Reversing an Array using Extra Array
* Reversing an Array using Swapping

### Shift

* Left Shift
* Right Shift

### Rotate

* Left Rotation
* Right Rotation

## Learning Outcomes

Through these implementations, I learned:

* How arrays are stored in memory
* Difference between size and length
* Array traversal techniques
* Insertion and deletion operations
* Searching algorithms and their efficiencies
* Array manipulation techniques such as reversing, shifting, and rotating
* Basic time complexity analysis of array operations
